Web Page Evaluation Assignment

Below are two web pages I evaluated—one appealing and one unappealing—followed by answers to the required questions.

Example 1: Appealing Web Page

a. URL of the website:

https://www.apple.com/

b. Does this page use tables?

The Apple homepage does not use tables for layout. Any tables that appear in the site code are used only for product specifications or structured data, not design.

c. Does this page use CSS?

Yes. CSS is used extensively for:

  • Page layout (grid-based, fully responsive)
  • Typography, spacing, and color
  • Brand consistency across the site
  • Mobile responsiveness

d. Is this page appealing? List three reasons.

  1. Clean and modern design with strong visual hierarchy.
  2. Consistent styling with balanced spacing and color usage.
  3. Highly responsive and optimized for different screen sizes.

e. Improvements (if any):

Although appealing, it could include improved accessibility features such as:

  • Optional dark mode toggle
  • Enhanced ARIA labels for assistive technologies
  • More descriptive alt text for decorative images

Example 2: Unappealing Web Page

a. URL of the website:

http://www.arngren.net/

b. Does this page use tables?

Yes. The page uses large nested HTML tables for the entire page layout, which is an outdated and inefficient design method.

c. Does this page use CSS?

Only minimal or outdated CSS is used. The site relies primarily on HTML attributes and images. CSS is not effectively used for layout, spacing, or styling.

d. Is this page appealing or unappealing? Provide three reasons.

  1. Cluttered layout with almost no whitespace.
  2. Lack of visual hierarchy—everything appears the same size.
  3. Confusing navigation with inconsistent text and image placement.

e. What would you do to improve this page?

  • Use CSS Flexbox or Grid instead of tables to create a clean layout.
  • Add consistent spacing, color themes, and typography.
  • Create a structured navigation menu with clear categories.
  • Optimize images and apply responsive mobile design.
